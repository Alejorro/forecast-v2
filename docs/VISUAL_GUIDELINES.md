# VISUAL_GUIDELINES.md

## Visual Direction

This is a professional internal business tool.

The design must feel:
- clean
- structured
- calm
- data-first

Avoid:
- dark dashboards
- neon colors
- gradients
- glassmorphism
- heavy shadows
- cluttered layouts
- decorative UI

---

## Layout Principles

- prioritize readability over visuals
- tables are the primary UI element
- charts support decisions, they do not dominate
- strong spacing between sections
- consistent alignment across all elements
- no cramped layouts

Forms:
- always use right-side drawers
- never navigate away from the main table

Navigation:
- top navigation always visible
- global year selector

---

## Color System

### Base

- background: #F8FAFC
- cards: #FFFFFF
- borders: #E2E8F0

### Text

- primary text: #0F172A
- secondary text: #64748B

### Functional Colors

- primary (actions, buttons): #2563EB
- success / won: #16A34A
- warning / pipeline: #F59E0B
- danger / negative gap: #DC2626
- neutral: #94A3B8

---

## Stage Colors

- Identified → neutral gray (soft background)
- Proposal 25 → light amber
- Proposal 50 → amber
- Proposal 75 → soft orange
- Won → green
- LOSS → muted gray

Badges:
- soft background color
- darker text
- rounded corners
- no strong borders

---

## Components

### Cards

- white background
- subtle border (#E2E8F0)
- soft shadow (very minimal)
- padding: 16–24px

### Tables (CRITICAL)

- high readability
- right-aligned numbers
- comfortable row height
- clear column spacing
- no cramped text

Avoid:
- dense rows
- too many separators
- unnecessary borders

---

## Typography

- clean sans-serif
- strong hierarchy

KPI numbers:
- large (text-2xl or larger)
- bold

Labels:
- small
- muted color

---

## Spacing

- section spacing: large (gap-6 or more)
- internal spacing: consistent
- never let elements touch each other visually

---

## Charts

- simple
- readable labels
- no clutter
- no unnecessary legends

Color usage:
- plan → neutral gray
- forecast → blue
- won → green

---

## Buttons

- primary → blue background
- secondary → outline or subtle gray
- destructive → red

---

## Overall Tone

Professional, minimal, structured.

The UI should feel like:
- a serious internal business tool
- not a marketing product
- not experimental design

If in doubt:
- use less color
- add more spacing
- simplify layout
